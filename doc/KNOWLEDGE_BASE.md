# Base de Conhecimento Integrada

## Objetivo

O comando `playtool learn` oferece explicações curtas, pesquisáveis e utilizáveis sem acesso ao navegador.

## Exemplos

```powershell
playtool learn mapping
playtool learn symbols
playtool learn versioncode
playtool learn screenshots
playtool learn review-url
```

## Estrutura de um tópico

Cada tópico deve conter:

- nome;
- resumo;
- por que importa;
- quando é necessário;
- localização ou exemplo;
- problemas comuns;
- comandos relacionados;
- referências oficiais.

## Tópicos iniciais

- app-bundle;
- version-code;
- version-name;
- mapping;
- native-symbols;
- tracks;
- release-notes;
- store-listing;
- screenshots;
- feature-graphic;
- icon;
- promotional-video;
- review-url;
- permissions;
- data-safety;
- app-links;
- service-account;
- edit;
- validate;
- commit;
- discard.

## Acessibilidade

O texto deve ser direto, sem tabelas complexas, sem desenhos e sem depender de links externos para explicar o conceito principal.
