# Roadmap

Este roadmap descreve a direção do projeto. Datas não são promessas e podem mudar conforme testes, contribuições e alterações nas APIs da Google Play.

## 1.3.0

- comando `playtool doctor`;
- diagnóstico do ambiente Python e credenciais;
- auditoria inicial de Manifest e permissões;
- validação de App Links e `assetlinks.json`;
- relatório único de prontidão para publicação;
- base do modo tutorial.

## 1.4.0

- tutorial guiado por etapas;
- retomada de sessão interrompida;
- mensagens otimizadas para JAWS, NVDA, VoiceOver e Orca;
- modo silencioso, modo detalhado e saída JSON consistente;
- testes de usabilidade com desenvolvedores cegos.

## 1.5.0

- suporte ampliado a listagens localizadas;
- geração de pacote completo de publicação;
- validação de política de privacidade e contatos;
- relatórios HTML acessíveis;
- integração opcional com Android Studio e Visual Studio Code.

## 2.0.0

- ação oficial para GitHub Actions;
- publicação contínua com aprovação manual;
- suporte a múltiplos aplicativos no mesmo repositório;
- sistema de plugins para novas lojas;
- API estável para integrações externas.

## Futuro

- F-Droid;
- Firebase App Distribution;
- Amazon Appstore;
- Microsoft Store;
- App Store Connect.
